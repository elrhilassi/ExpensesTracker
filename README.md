# Expenses Tracker

A full-stack project combining an Angular frontend and a .NET backend to manage and track expenses.

## Front-end


```bash
cd angular-frontend
npm install --f
ng serve
http://localhost:4200

```

## Back-end


```bash
cd angular-frontend
npm install --f
ng serve
http://localhost:4200

```

## Technologies Used

```python
cd dotnet-backend
dotnet restore
dotnet ef database update
dotnet run
http://localhost:5000

```
