# Expenses Tracker

A full-stack project combining an Angular frontend and a .NET backend to manage and track expenses.

## Installation


```bash
npm i -f 
```

## Technologies Used

```python
Frontend: Angular 18, TypeScript
Backend: .NET 8, Entity Framework Core
Database: postgres
Styling: Tailwind CSS
```
